# Veritas Consulting

Strona internetowa **Veritas Consulting** – prosty i elegancki projekt strony startowej dla firmy konsultingowej.

## Opis projektu
Strona zawiera:
- Sekcję powitalną
- Informacje o firmie
- Prezentację usług
- Formularz kontaktowy (link mailto)
- Responsywny design działający na komputerach i urządzeniach mobilnych

## Technologie
- HTML5
- CSS3
- JavaScript (opcjonalnie, do logów i interakcji)
